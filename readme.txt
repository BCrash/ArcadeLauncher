Game Dev Club Arcade Launcher - Made by Daniel Evans and Arthur Baney.

For each game you will need the following:
1 - folder containing the following files
2 - a title image saved as a png size 320x240
3 - an exe to launch the game
4 - a name.txt containing the title of the game
5 - a description.txt containing a description of your game
6 - a players.txt containing the amount of players e.g. "1-2 Players"
